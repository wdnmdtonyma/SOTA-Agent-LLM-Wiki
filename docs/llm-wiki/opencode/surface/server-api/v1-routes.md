## Gotchas

- V1 route declaration is Effect HttpApi, not Hono; adding a route belongs in `groups/*` plus `handlers/*`, not in an ad hoc router branchŃĆé[E: packages/opencode/src/server/routes/instance/httpapi/AGENTS.md:3][E: packages/opencode/src/server/routes/instance/httpapi/AGENTS.md:29]
- `/event` and `/global/event` are SSE route contracts even though OpenAPI needs manual normalization for `text/event-stream` in the legacy public specŃĆé[E: packages/opencode/src/server/routes/instance/httpapi/public.ts:155][E: packages/opencode/src/server/routes/instance/httpapi/public.ts:161]
