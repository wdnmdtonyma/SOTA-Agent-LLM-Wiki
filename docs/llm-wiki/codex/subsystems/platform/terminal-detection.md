## Detection flow

`detect_terminal_info_from_env` 先调用 `detect_multiplexer`，所以后续 terminal 分支都会携带 tmux/zellij metadata。[E: codex-rs/terminal-detection/src/lib.rs:301][E: codex-rs/terminal-detection/src/lib.rs:302] 如果存在非空 `TERM_PROGRAM`，只有 `TERM_PROGRAM=tmux`、multiplexer 是 tmux、且 tmux client info 能生成 terminal 时，才返回 tmux client terminal；否则普通非空 `TERM_PROGRAM` 分支直接返回并会 mask 后续 probes。[E: codex-rs/terminal-detection/src/lib.rs:304][E: codex-rs/terminal-detection/src/lib.rs:305][E: codex-rs/terminal-detection/src/lib.rs:306][E: codex-rs/terminal-detection/src/lib.rs:307][E: codex-rs/terminal-detection/src/lib.rs:308][E: codex-rs/terminal-detection/src/lib.rs:310][E: codex-rs/terminal-detection/src/lib.rs:313][E: codex-rs/terminal-detection/src/lib.rs:314][E: codex-rs/terminal-detection/src/lib.rs:315]

普通非空 `TERM_PROGRAM` 不存在时，detection 按 WEZTERM_VERSION、ITERM_SESSION_ID/ITERM_PROFILE/ITERM_PROFILE_NAME、TERM_SESSION_ID、KITTY_WINDOW_ID 或含 kitty 的 TERM、ALACRITTY_SOCKET 或 TERM=alacritty、KONSOLE_VERSION、GNOME_TERMINAL_SCREEN、VTE_VERSION、WT_SESSION、TERM fallback、unknown 的顺序返回。[E: codex-rs/terminal-detection/src/lib.rs:318][E: codex-rs/terminal-detection/src/lib.rs:323][E: codex-rs/terminal-detection/src/lib.rs:327][E: codex-rs/terminal-detection/src/lib.rs:335][E: codex-rs/terminal-detection/src/lib.rs:344][E: codex-rs/terminal-detection/src/lib.rs:357][E: codex-rs/terminal-detection/src/lib.rs:362][E: codex-rs/terminal-detection/src/lib.rs:370][E: codex-rs/terminal-detection/src/lib.rs:375][E: codex-rs/terminal-detection/src/lib.rs:383][E: codex-rs/terminal-detection/src/lib.rs:387]

`detect_multiplexer` 通过非空 `TMUX` 或 `TMUX_PANE` 识别 tmux，通过非空 `ZELLIJ`、`ZELLIJ_SESSION_NAME` 或 `ZELLIJ_VERSION` 识别 zellij；tmux/zellij version 都写入各自 optional version 字段，process env path 的 zellij version 会在 env var 缺失时调用 command fallback，都没有 multiplexer env 时返回 `None`。[E: codex-rs/terminal-detection/src/lib.rs:390][E: codex-rs/terminal-detection/src/lib.rs:391][E: codex-rs/terminal-detection/src/lib.rs:392][E: codex-rs/terminal-detection/src/lib.rs:393][E: codex-rs/terminal-detection/src/lib.rs:397][E: codex-rs/terminal-detection/src/lib.rs:398][E: codex-rs/terminal-detection/src/lib.rs:399][E: codex-rs/terminal-detection/src/lib.rs:401][E: codex-rs/terminal-detection/src/lib.rs:402][E: codex-rs/terminal-detection/src/lib.rs:269][E: codex-rs/terminal-detection/src/lib.rs:270][E: codex-rs/terminal-detection/src/lib.rs:271][E: codex-rs/terminal-detection/src/lib.rs:474][E: codex-rs/terminal-detection/src/lib.rs:477]

tmux client helper executes `tmux display-message -p #{client_termtype}` and `tmux display-message -p #{client_termname}`; `terminal_from_tmux_client_info` prioritizes splitting termtype into program/version and uses termname as the TERM capability string, falling back to termname-only terminal info only when termtype is absent.[E: codex-rs/terminal-detection/src/lib.rs:453][E: codex-rs/terminal-detection/src/lib.rs:454][E: codex-rs/terminal-detection/src/lib.rs:455][E: codex-rs/terminal-detection/src/lib.rs:457][E: codex-rs/terminal-detection/src/lib.rs:460][E: codex-rs/terminal-detection/src/lib.rs:461][E: codex-rs/terminal-detection/src/lib.rs:462][E: codex-rs/terminal-detection/src/lib.rs:463][E: codex-rs/terminal-detection/src/lib.rs:413][E: codex-rs/terminal-detection/src/lib.rs:417][E: codex-rs/terminal-detection/src/lib.rs:418][E: codex-rs/terminal-detection/src/lib.rs:420][E: codex-rs/terminal-detection/src/lib.rs:421][E: codex-rs/terminal-detection/src/lib.rs:423][E: codex-rs/terminal-detection/src/lib.rs:427][E: codex-rs/terminal-detection/src/lib.rs:432][E: codex-rs/terminal-detection/src/lib.rs:434]

## User-Agent formatting

`TerminalInfo::user_agent_token` 优先使用 `term_program[/version]`，没有 term_program 时使用非空 `term`，再 fallback 到 `TerminalName`-specific token；最后统一调用 `sanitize_header_value`。[E: codex-rs/terminal-detection/src/lib.rs:176][E: codex-rs/terminal-detection/src/lib.rs:177][E: codex-rs/terminal-detection/src/lib.rs:178][E: codex-rs/terminal-detection/src/lib.rs:180][E: codex-rs/terminal-detection/src/lib.rs:183][E: codex-rs/terminal-detection/src/lib.rs:186][E: codex-rs/terminal-detection/src/lib.rs:208] `sanitize_header_value` 把不属于 ASCII alphanumeric、`-`、`_`、`.`、`/` 的字符替换为 `_`。[E: codex-rs/terminal-detection/src/lib.rs:500][E: codex-rs/terminal-detection/src/lib.rs:502][E: codex-rs/terminal-detection/src/lib.rs:503][E: codex-rs/terminal-detection/src/lib.rs:504][E: codex-rs/terminal-detection/src/lib.rs:507][E: codex-rs/terminal-detection/src/lib.rs:508][E: codex-rs/terminal-detection/src/lib.rs:509]

`terminal_name_from_term_program` 会 trim、删除空格/连字符/下划线/点号并转小写后匹配 known terminal names；`format_terminal_version` 才是 name/version formatting helper。[E: codex-rs/terminal-detection/src/lib.rs:512][E: codex-rs/terminal-detection/src/lib.rs:513][E: codex-rs/terminal-detection/src/lib.rs:514][E: codex-rs/terminal-detection/src/lib.rs:515][E: codex-rs/terminal-detection/src/lib.rs:516][E: codex-rs/terminal-detection/src/lib.rs:517][E: codex-rs/terminal-detection/src/lib.rs:520][E: codex-rs/terminal-detection/src/lib.rs:521][E: codex-rs/terminal-detection/src/lib.rs:523][E: codex-rs/terminal-detection/src/lib.rs:525][E: codex-rs/terminal-detection/src/lib.rs:527][E: codex-rs/terminal-detection/src/lib.rs:529][E: codex-rs/terminal-detection/src/lib.rs:533][E: codex-rs/terminal-detection/src/lib.rs:538][E: codex-rs/terminal-detection/src/lib.rs:539][E: codex-rs/terminal-detection/src/lib.rs:540][E: codex-rs/terminal-detection/src/lib.rs:541]

## 设计动机与权衡

terminal detection 优先使用非空 `TERM_PROGRAM` 和显式 vendor env vars，再 fallback 到 generic `TERM`，是为了在现代 terminal 中得到更具体的 product token，同时保留未知环境的可用性。[I] 该结论由 `detect_terminal_info_from_env` 分支顺序共同支撑。[E: codex-rs/terminal-detection/src/lib.rs:301][E: codex-rs/terminal-detection/src/lib.rs:304][E: codex-rs/terminal-detection/src/lib.rs:318][E: codex-rs/terminal-detection/src/lib.rs:323][E: codex-rs/terminal-detection/src/lib.rs:335][E: codex-rs/terminal-detection/src/lib.rs:383][E: codex-rs/terminal-detection/src/lib.rs:387]

tmux client term metadata 只在非空 `TERM_PROGRAM=tmux` 且 tmux multiplexer active 时覆盖 terminal identity，说明实现想把 tmux session 归因到 underlying terminal，但避免在普通 shell 或非 tmux `TERM_PROGRAM` 下执行 tmux-specific interpretation。[I] 该结论由 `TERM_PROGRAM=tmux` guard 和 tmux client info branch 支撑。[E: codex-rs/terminal-detection/src/lib.rs:304][E: codex-rs/terminal-detection/src/lib.rs:305][E: codex-rs/terminal-detection/src/lib.rs:306][E: codex-rs/terminal-detection/src/lib.rs:307][E: codex-rs/terminal-detection/src/lib.rs:308][E: codex-rs/terminal-detection/src/lib.rs:310][E: codex-rs/terminal-detection/src/lib.rs:313][E: codex-rs/terminal-detection/src/lib.rs:315]

## Gotchas

- 非空 `TERM_PROGRAM` 会 mask WEZTERM、WT_SESSION 等后续 probes，除非 `TERM_PROGRAM=tmux` 且 tmux client info 可用。[E: codex-rs/terminal-detection/src/lib.rs:304][E: codex-rs/terminal-detection/src/lib.rs:305][E: codex-rs/terminal-detection/src/lib.rs:306][E: codex-rs/terminal-detection/src/lib.rs:307][E: codex-rs/terminal-detection/src/lib.rs:308][E: codex-rs/terminal-detection/src/lib.rs:310][E: codex-rs/terminal-detection/src/lib.rs:313][E: codex-rs/terminal-detection/src/lib.rs:315]
- `TerminalName::Unknown` 不等于没有 terminal metadata；`TERM` fallback 会产生 `TerminalInfo::from_term(term, multiplexer)`，tmux termtype 也可能映射成 Unknown name 但保留 term_program/term。[E: codex-rs/terminal-detection/src/lib.rs:383][E: codex-rs/terminal-detection/src/lib.rs:384][E: codex-rs/terminal-detection/src/lib.rs:420][E: codex-rs/terminal-detection/src/lib.rs:422][E: codex-rs/terminal-detection/src/lib.rs:423][E: codex-rs/terminal-detection/src/lib.rs:432][E: codex-rs/terminal-detection/src/lib.rs:434]
- public `terminal_info()` 使用 `OnceLock` cache；同一 process 里修改 env 后不会刷新 cached result。[I] 该结论由 one-time `get_or_init` cache 语义支撑。[E: codex-rs/terminal-detection/src/lib.rs:217][E: codex-rs/terminal-detection/src/lib.rs:281][E: codex-rs/terminal-detection/src/lib.rs:282][E: codex-rs/terminal-detection/src/lib.rs:283]
