> `ui.permissions` 是 Claude Code 对 tool permission、file permission、shell permission、permission rules 和 bypass mode 做人工确认的 Ink UI 组件族。[I]

## 能回答的问题
- `PermissionRequest` 怎样按 tool 路由到专用 request component?
- Bash、PowerShell、file、filesystem 和 rules UI 分别在哪些组件中渲染?
- permission UI 怎样调用 `toolUseConfirm` callback、写 AppState 或写 settings?

## 族干什么
`PermissionRequest` 的路由 key 是 `toolUseConfirm.tool`,并通过 `permissionComponentForTool` 的 `switch (tool)` 选择组件。[E: components/permissions/PermissionRequest.tsx:47][E: components/permissions/PermissionRequest.tsx:48][E: components/permissions/PermissionRequest.tsx:193] FileEdit 返回 `FileEditPermissionRequest`,FileWrite 返回 `FileWritePermissionRequest`,Bash 返回 `BashPermissionRequest`,PowerShell 返回 `PowerShellPermissionRequest`,Glob/Grep/FileRead 返回 `FilesystemPermissionRequest`,默认返回 `FallbackPermissionRequest`。[E: components/permissions/PermissionRequest.tsx:49][E: components/permissions/PermissionRequest.tsx:50][E: components/permissions/PermissionRequest.tsx:51][E: components/permissions/PermissionRequest.tsx:52][E: components/permissions/PermissionRequest.tsx:53][E: components/permissions/PermissionRequest.tsx:54][E: components/permissions/PermissionRequest.tsx:55][E: components/permissions/PermissionRequest.tsx:56][E: components/permissions/PermissionRequest.tsx:75][E: components/permissions/PermissionRequest.tsx:76][E: components/permissions/PermissionRequest.tsx:77][E: components/permissions/PermissionRequest.tsx:78][E: components/permissions/PermissionRequest.tsx:80] `ToolUseConfirm` 提供 `onAllow`、`onReject` 和 `recheckPermission`,而 `PermissionRequest` 把整个 `toolUseConfirm` 传给被选中的 `PermissionComponent`。[E: components/permissions/PermissionRequest.tsx:124][E: components/permissions/PermissionRequest.tsx:125][E: components/permissions/PermissionRequest.tsx:126][E: components/permissions/PermissionRequest.tsx:202]

## 巨型组件深挖
