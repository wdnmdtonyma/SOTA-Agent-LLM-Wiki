> `ref.ai.core-types` 是 `packages/ai/src/types.ts` exported 类型的字段级目录:覆盖模型元数据、请求上下文、消息 union、assistant 流事件、usage/cost 与 image/compat 辅助类型。

## 能回答的问题

- `Model`、`Context`、`Message`、`AssistantMessage`、`AssistantMessageEvent`、`Usage` 各自有哪些字段?
- `Message` 的三种 role 形态如何把 user、assistant、tool result 串成统一历史?
- provider stream API、simple stream API、image generation API 分别消费哪些上下文和 option 类型?
- `AssistantMessageEvent` 的 partial event 与 terminal event 如何对应 `AssistantMessageEventStream`?
- 模型能力、thinking level、prompt cache、compat override 在 `types.ts` 中落在哪些类型上?
