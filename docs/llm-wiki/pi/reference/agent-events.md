> `ref.agent.agent-events` 是 agent-core 事件 union 的字段级目录:覆盖 `AgentEvent` 的 10 个 core runtime variant、`AgentHarnessEvent` 叠加的 harness-owned event interfaces、hook result map,以及这些 event payload/result 直接引用的 exported 支撑类型。

## 能回答的问题

- `AgentEvent` 当前有哪些 lifecycle、turn、message、tool execution variant?
- `AgentHarnessEvent` 在 `AgentEvent` 之外额外公开哪些 harness-owned event?
- 每个 event 的 `type` discriminator、payload 字段和字段类型是什么?
- 哪些 harness events 允许 hook handler 返回结果,返回类型分别是什么?
- event payload 直接引用的 `AgentHarnessResources`、stream option patch、compaction/tree preparation 类型各自包含哪些字段?
