## Ground Truth

`components/index.ts` 是 extension-facing barrel: 它 re-export `ArminComponent`、message components、selector/dialog components、`FooterComponent`、keybinding hint helpers、`renderDiff` 和 `truncateToVisualLines` 等符号 [E: packages/coding-agent/src/modes/interactive/components/index.ts:2] [E: packages/coding-agent/src/modes/interactive/components/index.ts:38]。`packages/coding-agent/src/index.ts` 再把这些 UI components 从 package public API 导出, 供 custom tools/extensions 复用 [E: packages/coding-agent/src/index.ts:327] [E: packages/coding-agent/src/index.ts:364]。

`InteractiveMode` 直接 import 大多数 session-bound components, 包括 `ArminComponent`、message components、`CountdownTimer`、`CustomEditor`、selector/dialog components、`FooterComponent` 和 `ToolExecutionComponent` [E: packages/coding-agent/src/modes/interactive/interactive-mode.ts:100] [E: packages/coding-agent/src/modes/interactive/interactive-mode.ts:128]。startup UI 复用 `ExtensionSelectorComponent`、`FirstTimeSetupComponent` 和 `ExtensionInputComponent`, 在进入长期 interactive session 前临时创建 TUI [E: packages/coding-agent/src/cli/startup-ui.ts:134] [E: packages/coding-agent/src/cli/startup-ui.ts:237]。`pi config` 命令直接 import `ConfigSelectorComponent`, 但该 component 不经过 `components/index.ts` barrel [E: packages/coding-agent/src/cli/config-selector.ts:8] [E: packages/coding-agent/src/cli/config-selector.ts:27] [U]。

本 catalog 的实例粒度按目录文件展开: 38 个 `.ts` 文件全部出现, 其中 `index.ts` 是 barrel 文件而不是 runtime component class [I]。如果未来把 catalog instance_count 改成“public exported component/helper 数”, `index.ts` 是否算作实例需要重新定义 [U]。

## 分类视图

- Message/render components: `AssistantMessageComponent`, `UserMessageComponent`, `CustomMessageComponent`, `BranchSummaryMessageComponent`, `CompactionSummaryMessageComponent`, `SkillInvocationMessageComponent`, `ToolExecutionComponent`, `BashExecutionComponent` [I]。
- Selector/dialog components: `SettingsSelectorComponent`, `ModelSelectorComponent`, `ScopedModelsSelectorComponent`, `SessionSelectorComponent`, `TreeSelectorComponent`, `UserMessageSelectorComponent`, `TrustSelectorComponent`, `OAuthSelectorComponent`, `LoginDialogComponent`, `ExtensionSelectorComponent`, `ExtensionInputComponent`, `ExtensionEditorComponent`, `FirstTimeSetupComponent`, `ConfigSelectorComponent`, `ShowImagesSelectorComponent`, `ThemeSelectorComponent`, `ThinkingSelectorComponent` [I]。
- Structural/helper components: `CustomEditor`, `FooterComponent`, `DynamicBorder`, `BorderedLoader`, `CountdownTimer`, `keybinding-hints.ts`, `renderDiff`, `truncateToVisualLines`, `session-selector-search.ts` [I]。
- Announcement/animation components: `ArminComponent`, `DaxnutsComponent`, `EarendilAnnouncementComponent` [I]。

## 不确定项

- `components/index.ts` 是 public extension-facing barrel, 但 catalog group 的 ground truth 是整个 directory;因此 `index.ts` 是否应被计为一个 catalog instance, 还是只应作为 barrel metadata, 当前只能按目录文件数纳入 [U]。
- `ConfigSelectorComponent`、`CountdownTimer`、`EarendilAnnouncementComponent`、`session-selector-search.ts` 有真实调用方但不从 `components/index.ts` 导出;这看起来是内部-only 边界, 但是否有意不对 extension 公开需要 L2/maintainer 判断 [U]。
- `ShowImagesSelectorComponent`、`ThemeSelectorComponent`、`ThinkingSelectorComponent` 仍从 public API 导出, 但当前 main interactive mode 没有直接 import;它们可能是 extension compatibility surface 或残留 UI, 需要后续按 release/API policy 判断 [U]。
